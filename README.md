## Archivo gitignore
El archivo .gitignore se usa en Git para indicar qué archivos o carpetas NO deben subirse al repositorio en GitHub (o cualquier repo).

### ¿Por qué es conveniente incluirlo?
- Evita archivos inescesarios
- mantiene el repositorio limpio
- evita subir archivos sensibles

### ¿Cuándo se debe hacer?
Lo ideal es crearlo desde el inicio del proyecto.
Pero tambien se puede hacerlo despues, solamente hay que tener en cuenta que luego habrá que borrar archivos manualmente.

### ¿Cómo configuraría el archivo .gitignore?
Es un archivo de texto donde escribis reglas.
- .exe
- .o
- .log
